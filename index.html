Lineage-OS-Installer

A graphical installer for the open-source Android firmware replacement LineageOS. It aims to make installing
the ROM as easy and simple as possible, guiding the user through every step of what they need to do. In addition to being a graphical guide, it:

- Automatically unlocks the bootloader
- Automatically flashes TWRP Recovery
- Wipes, backs up, and flashes `lineageos.zip` (LineageOS) and `gapps.zip` (Open GApps) with out the user interacting with the TWRP Recovery Interface.
- The user does not need to type a single command to flash the ROM; it is all handled by automated scripts.
